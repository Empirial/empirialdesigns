# Empirial Designs - Project Goals & Roadmap

## 🌟 The Vision
To build **Empirial Designs**, an AI-powered "Super Engineer" web application platform that functions like Lovable.ai. 
It allows users to generate full-stack web applications entirely through natural language prompts, instantly visualizing and iteratively editing the results in a live workspace.

## 🎯 Core Objectives & Features

1. **The AI Generation Engine**
   - Accept user prompts and stream "thought processes" and real-time code generation.
   - Generate production-ready code (React, Tailwind CSS, Vite dependencies).

2. **The Workspace UI (The Editor)**
   - **Left Pane (Chat & Control):** Maintain a persistent chat history, display AI reasoning, and provide a robust prompt input for iterative changes.
   - **Right Pane (Live Canvas):** A dynamic iframe that instantly compiles and renders the code generated by the AI in real-time.
   - **Visual Inspector (Click-to-Edit):** Implement a cross-frame selection tool so users can visually click on elements inside the live canvas to tell the AI exactly what to modify.

3. **Backend & Infrastructure**
   - Connect frontend chat requests seamlessly to Firebase Cloud Functions and LLMs.
   - Manage user authentication, save workspace states, and handle project routing (e.g., `/preview/:repoId`).

4. **Brand Aesthetic**
   - Maintain the premium **Empirial Gold & Amber** theme (`hsl(45 100% 51%)`) across all complex dark-mode interfaces (`#09090b`), replacing standard boilerplate color palettes.
